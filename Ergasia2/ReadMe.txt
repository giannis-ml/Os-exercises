ΑΜ: 1115201900106

Στην δεύτερη εργασία, έχω υλοποιήσει ένα πρόγραμμα το οποίο διαβάσει το binary file και αποθηκεύει τα δεδομένα του σε ένα πίνακα
απο struct τύπου MyRecord. Επίσης καταστευάζει το δέντρο με την ιεραρχία κλάσεων που ζητείται χρησιμοποιώντας fork.
Τέλος καταστευάζει τα pipe που χρειάζονται για να επικοινωνεί η ρίζα με τα ενδιάμεσα παιδιά, προκειμένου να τους περάσει σωστά τις
εγγραφές. Υπολογίζεται στο κομμάτι του γονέα το πόσες εγγραφές θα περνούνται σε κάθε παιδί. Η εργασία δεν είναι ολοκληρωμένη.
Κυρίως λόγω φορτωμένου προγράμματος. Έκανα το καλύτερο που μπορούσα για να δείξω οτι είμαι σε ένα βαθμό εξοικειωμένος με
τις κλήσεις συστήματος, την κατασκευή και τον χειρισμό διεργασιών.
